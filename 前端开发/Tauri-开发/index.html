Tauri 开发

Tauri 是一个类似于 Electron 的跨平台 Web 开发方案，使用操作系统的原生 WebView 渲染、Rust 作为后台开发语言。

开发环境配置

Rust

工具下载地址验证命令
RustRust Installrustc –version
Cargo-cargo –vresion
Rust RoverRust Rover-

MSVC

Windows 下需要 MSVC。

从网站或者 winget install Microsoft.VisualStudio.2022.BuildTools --override "--add Microsoft.VisualStudio.Workload.VCTools" 安装。

Tauri

cargo install tauri-cli
npm install -g pnpm

项目构建

pnpm create tauri-app@latest
pnpm tauri init
pnpm tauri dev
pnpm tauri build
